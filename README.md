# VibeCoding 2 - Procedural Generation Experiments

Experimental version of VibeCoding exploring procedural music generation.

## About

This is a research/development version separate from the production VibeCoding site. It starts with the Phase 1 foundation and explores procedural generation techniques for creating dynamic, generative music.

## Original VibeCoding

The stable, production version is at:
- **Live Site**: https://vibecoding.gostnode.com
- **Repo**: https://github.com/Gregypo085/vibecoding

## Phase 2 Goals

Exploring procedural generation approaches for music:
- Algorithmic composition
- Dynamic pattern generation
- Rule-based music creation
- Real-time generative systems

## Development

To run locally:
```bash
python3 -m http.server 8000
```

Then open http://localhost:8000
